# TrieTree 字典树

## Trie

### 使用场景

统计和排序大量的字符串，搜索引擎系统用于文本词频统计

### 优点

最大限度地减少无谓的字符串比较，查询效率比哈希表高。

### 性质

- 根节点不包含字符，除根节点外每一个节点都只包含一个字符。
- 从根节点到某一节点，路径上经过的字符连接起来，为该节点对应的字符串。
- 每个节点的所有子节点包含的字符都不相同。

### 结构

```c
typedef struct TrieNode                     //Trie结点声明  
{ 
    bool isStr;                            //标记该结点处是否构成单词  
    struct TrieNode *next[MAX];            //儿子分支  
}Trie; 
```
